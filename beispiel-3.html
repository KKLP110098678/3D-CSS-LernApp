Beispiel 3: Parallax-Layer mit Tiefe

Verwendungszweck: Hero-Abschnitte und visuelle Storytelling-Bereiche, die räumliche Tiefe mit mehreren Ebenen benötigen.

HTML-Struktur (Container-Hierarchie)

  • section.card
  • div.demo.demo-parallax
    • div.scene
      • div.layer.layer-back
      • div.layer.layer-mid
      • div.layer.layer-front

Container-Erklärung

  • section.card: Inhaltlicher Rahmen für ein Beispiel in der Hauptseite. Zum Code
  • div.demo: Einheitlicher Demo-Container als Bühne. Zum Code
  • div.demo-parallax: Zentriert die Parallax-Szene innerhalb der Bühne. Zum Code
  • div.scene: 3D-Kontext mit perspective und preserve-3d für verschachtelte Ebenen. Zum Code
  • div.layer: Gemeinsame Basiseigenschaften aller Ebenen (absolute Position, Zentrierung). Zum Code
  • .layer-back: Hintergrundebene bei negativem Z-Wert. Zum Code
  • .layer-mid: Mittlere Referenzebene nahe Z = 0. Zum Code
  • .layer-front: Vordergrundebene mit positivem Z-Wert für Nähe-Effekt. Zum Code

Quellcode

<!-- HTML -->
<section class="card">
  <div class="demo demo-parallax">
    <div class="scene">
      <div class="layer layer-back">Hintergrund</div>
      <div class="layer layer-mid">Mitte</div>
      <div class="layer layer-front">Vorne</div>
    </div>
  </div>
</section>

/* CSS */
.scene {
  perspective: 500px;
  transform-style: preserve-3d;
}

.layer-back {
  transform: translate3d(0, 0, -40px);
}

.layer-mid {
  transform: translate3d(0, -2px, 0);
}

.layer-front {
  transform: translate3d(0, -6px, 35px);
}
