Beispiel 2: Off-Canvas Menü

Verwendungszweck: Mobile Navigation oder Filterleisten, die platzsparend seitlich ein- und ausgeblendet werden.

HTML-Struktur (Container-Hierarchie)

  • section.card
  • div.demo.demo-drawer
    • button#toggleDrawer.btn
    • div#drawerWrap.drawer-wrap
      • aside#drawer.drawer
        • h3 und ul

Container-Erklärung

  • section.card: Umschließt das komplette Beispiel in der Übersicht. Zum Code
  • div.demo: Standard-Demo-Container mit konsistentem Layout. Zum Code
  • div.demo-drawer: Semantischer Marker für das Drawer-Beispiel. Zum Code
  • button.btn: UI-Trigger; schaltet die Klasse open am Drawer. Zum Code
  • div.drawer-wrap: Sichtfenster mit overflow: hidden, damit das Panel außerhalb abgeschnitten bleibt. Zum Code
  • aside.drawer: Das verschiebbare Panel; startet mit translate3d(-190px, 0, 0) außerhalb links. Zum Code
  • aside.drawer.open: Endzustand im Sichtbereich mit translate3d(0, 0, 0). Zum Code
  • .drawer h3 / .drawer ul: Struktur und Lesbarkeit innerhalb des Menüs. Zum Code

Quellcode

<!-- HTML -->
<section class="card">
  <div class="demo demo-drawer">
    <button id="toggleDrawer" class="btn">Menü umschalten</button>
    <div class="drawer-wrap">
      <aside class="drawer">
        <h3>Menü</h3>
        <ul>...</ul>
      </aside>
    </div>
  </div>
</section>

/* CSS */
.drawer {
  transform: translate3d(-190px, 0, 0);
}

.drawer.open {
  transform: translate3d(0, 0, 0);
}

/* JavaScript */
toggleDrawerButton.addEventListener("click", () => {
  drawer.classList.toggle("open");
});
