Beispiel 4: Performance-freundliche Bewegungsanimation

Verwendungszweck: Micro-Animationen (z. B. Loader), die auf vielen Geräten flüssig bleiben sollen.

HTML-Struktur (Container-Hierarchie)

  • section.card
  • div.demo.demo-motion
    • div.track
      • div.dot

Container-Erklärung

  • section.card: Kapselt das Beispiel in der Übersicht. Zum Code
  • div.demo: Einheitlicher Umgebungscontainer. Zum Code
  • div.demo-motion: Semantischer Marker für das Bewegungsbeispiel. Zum Code
  • div.track: Definiert die sichtbare Laufstrecke des bewegten Elements. Zum Code
  • div.dot: Das animierte Objekt; bewegt sich per @keyframes slide3d. Zum Code
  • @keyframes slide3d: Hauptanimation mit 3D-Translation auf X/Y/Z. Zum Code
  • @media (max-width: 600px) + slide3d-mobile: Angepasste Bewegungsdistanz auf kleinen Displays. Zum Code

Quellcode

<!-- HTML -->
<section class="card">
  <div class="demo demo-motion">
    <div class="track">
      <div class="dot"></div>
    </div>
  </div>
</section>

/* CSS */
@keyframes slide3d {
  0% { transform: translate3d(0, 0, 0); }
  50% { transform: translate3d(300px, -6px, 30px); }
  100% { transform: translate3d(0, 0, 0); }
}

@media (max-width: 600px) {
  .dot { animation-name: slide3d-mobile; }
}
