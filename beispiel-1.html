Beispiel 1: Hover-Karte mit Tiefeneffekt

Verwendungszweck: Interaktive Cards/Buttons, die beim Hover eine räumliche Rückmeldung geben und dadurch klickbarer wirken.

HTML-Struktur (Container-Hierarchie)

  • section.card
  • div.demo.demo-card
    • div.product-card
      • h3 und p

Container-Erklärung

  • section.card: Kapselt das komplette Beispiel als eigenständigen Inhaltsblock. Zum Code
  • div.demo: Allgemeiner Demo-Rahmen (Innenabstand, Hintergrund, Rahmen). Zum Code
  • div.demo-card: Definiert den 3D-Raum per perspective, damit Z-Bewegung sichtbar wird. Zum Code
  • div.product-card: Das eigentliche animierte Element mit Startzustand und Transition. Zum Code
  • div.product-card:hover: Hebt das Element mit translate3d(0, -10px, 24px) an (Y nach oben, Z nach vorne). Zum Code
  • .product-card h3 / .product-card p: Sorgt für klare Typografie innerhalb der Card. Zum Code

Quellcode

<!-- HTML -->
<section class="card">
  <div class="demo demo-card">
    <div class="product-card">
      <h3>Produktkarte</h3>
      <p>Hover für 3D-Lift</p>
    </div>
  </div>
</section>

/* CSS */
.product-card {
  transform: translate3d(0, 0, 0);
  transition: transform 220ms ease, box-shadow 220ms ease;
}

.product-card:hover {
  transform: translate3d(0, -10px, 24px);
}

.product-card h3,
.product-card p {
  margin: 0;
}
